- React

# 주요 기능

- 회원가입 및 로그인, 로그아웃 (jwt)
- 로그인을 통해 인증된 유저만 게시판 기능 사용 가능
- 세부적으로 게시물 업로드, 위시리스트, 댓글 기능 가능
- 인증 된 유저의 마이페이지에서 본인 위시리스트 확인 및 메인페이지에서 위시등록 및 삭제 가능
- 게시물 작성 페이지에서 이미지를 포함한 게시물 업로드 (및 수정) 가능
- 게시물 상세 페이지에서 댓글 작성 및 댓글 리스트 확인 가능
